## 注意事項

### Gemini APIレート制限について

API呼び出しを伴うテスト（※API依存）は、Gemini APIの無料枠制限（1分15リクエスト）の影響を受けます。

**対策**:
- API連携テストは `test.describe.serial` でシリアル実行
- 各テスト前に2秒の待機時間を設定
- レート制限に達した場合は、数分待機してから再実行

**テストコマンド**:
```bash
# 基本テストのみ実行（API不要）
npx playwright test --grep "基本機能"

# API連携テスト実行（シリアル）
npx playwright test --grep "API連携" --workers=1
```
